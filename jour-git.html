Skills Definition
Gif Clone Il permet de cloner un dossier et le lier depuis git hub
Git commited Permet de précharger un fichier pour l'envoyer sur le github
Git pull Permet d'extraire un fichier de la branche du github
git push permet d'extraire un fichier du github et de fusionner cette version à celle du fichier local
Git merge Permet de fussioner les dossiers correspondant
Git Checkout Permet de switcher entre les branches
